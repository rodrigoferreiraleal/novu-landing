Política de Privacidade

Última actualização: Março de 2026

A NOVU respeita a tua privacidade. Esta política explica que dados recolhemos, como os usamos e os teus direitos ao abrigo do RGPD (Regulamento Geral sobre a Protecção de Dados — UE 2016/679).

1. Quem somos

A NOVU é uma plataforma de informação e orientação para imigrantes em Portugal, operada a título pessoal. Contacto: geral@novuai.pt

2. Que dados recolhemos

  • Nome e email — quando te registas para acesso ao assistente ou lista de espera
  • Idioma de preferência — para personalizar a experiência
  • Mensagens enviadas ao assistente — processadas pela API da Anthropic para gerar respostas
  • Dados de utilização — número de perguntas, timestamps (através de cookies)
  • Endereço IP — para controlo de taxa de utilização (rate limiting)

3. Como usamos os dados

  • Prestar o serviço de assistência via chat
  • Enviar comunicações sobre o produto (apenas com consentimento)
  • Melhorar a qualidade das respostas e a experiência do utilizador
  • Cumprir obrigações legais

4. Base legal do tratamento (RGPD)

  • Consentimento (Art. 6.º, n.º 1, al. a)) — para o registo de email e comunicações de marketing
  • Execução de contrato (Art. 6.º, n.º 1, al. b)) — para prestar o serviço solicitado
  • Interesse legítimo (Art. 6.º, n.º 1, al. f)) — para segurança e melhoria do serviço

5. Subcontratantes (terceiros)

Os teus dados podem ser processados pelos seguintes subcontratantes:

  • Anthropic (EUA) — processamento das mensagens do chat via API. Política: anthropic.com/privacy
  • Supabase (UE — Ireland) — base de dados para armazenamento de emails registados
  • Vercel (EUA) — alojamento do site e APIs
  • Formspree — reencaminhamento de emails de contacto
  • Google Analytics — estatísticas de utilização anónimas

As transferências para fora da UE (Anthropic, Vercel) são efectuadas com garantias adequadas (Cláusulas Contratuais Padrão).

6. Retenção de dados

  • Emails registados: conservados enquanto o serviço estiver activo ou até pedido de eliminação
  • Mensagens do chat: não são armazenadas permanentemente pela NOVU (processadas em tempo real)
  • Cookies: expiram em 24 horas (sessão) ou 30 dias (preferências)

7. Os teus direitos (RGPD)

Tens os seguintes direitos relativamente aos teus dados pessoais:

  • Acesso — pedir uma cópia dos teus dados
  • Rectificação — corrigir dados incorrectos
  • Eliminação ("direito ao esquecimento") — pedir a eliminação dos teus dados
  • Portabilidade — receber os teus dados num formato estruturado
  • Oposição — opor-te ao tratamento para fins de marketing
  • Retirada de consentimento — a qualquer momento, sem afectar o tratamento anterior

Para exercer os teus direitos, envia um email para geral@novuai.pt. Respondemos em até 30 dias.

8. Cookies

Utilizamos os seguintes tipos de cookies:

  • Essenciais — necessários para o funcionamento do site (sessão, preferências de idioma, contador de perguntas)
  • Analíticos — Google Analytics (anónimo) para compreender como o site é utilizado

Podes gerir as tuas preferências de cookies a qualquer momento através do banner de cookies.

9. Segurança

Adoptamos medidas técnicas e organizacionais para proteger os teus dados, incluindo encriptação HTTPS, acesso restrito à base de dados e uso de fornecedores certificados.

10. Reclamações

Tens o direito de apresentar reclamação à autoridade de controlo competente em Portugal: CNPD (Comissão Nacional de Protecção de Dados) — cnpd.pt

11. Alterações a esta política

Podemos actualizar esta política periodicamente. A data de "última actualização" no topo indica quando foi revista pela última vez. Alterações significativas serão comunicadas por email aos utilizadores registados.

Contacto

Para qualquer questão sobre privacidade: geral@novuai.pt
